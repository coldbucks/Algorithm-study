# 알려주지. 멘붕에서 해설을 보았다.. 이런식으로 알려주는구나. 한수 배운다.
# 10번 따라 치고, 써보기 실시

N = int(input())

n = 0

M = N

while True:
    
    a = M//10
    
    b = M%10
    
    c = (a+b)%10
    
    e = (10*b)+c
    
    n += 1
    if e == N:
        break
    
    M = e

print(n)
