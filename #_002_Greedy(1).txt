# 그리디 알고리즘
# 현재 상황에서 지금 당장 좋은 것만 고르는 방법
# 문제 유형이 다양하기 때문에 암기한다고 해서 항상 잘 풀수 있는 알고리즘 유형이 아니다
# 많은 유형을 접해보고 문제를 풀어보며 훈련을 해야한다.


#거스름돈 

n = 1260
count = 0

money = [500,100,50,10]

for i in money:
    count += n // i
    n %= i

print(count)

# 코딩문제-> 문제해결 아이디어 탐색(그리드알고리즘, 다이나믹프로그래밍, 그래프 알고리즘)

# 이름 수정

n = 1260
count = 0

coin_types = [500,100,50,10]

for coin in coin_types:
    count += n // coin
    n %= coin

print(count)
